# Travel Plan Server

## Getting Started

### Prerequisites

Add .env to `/server` folder

```bash
GOOGLE_CLIENT_ID=<YOUR_GOOGLE_CLIENT_ID>
GOOGLE_CLIENT_SECRET=<YOUR_GOOGLE_CLIENT_SECRET>
```

### Install Dependencies

```bash
yarn install
```

### Start Server

```bash
yarn dev
```

### Start Server in Production Mode

```bash
yarn build
```

Then

```bash
yarn start
```
